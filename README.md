# Assignment- 6

## Project name: DigiTools

### Description: An online platform for digital tools selling

## ⚙️ Technologies

- React.js
- Tailwind CSS, DaisyUI
- React-Toastify

## Core features

1. **Easy purchase:** User can quickly browse, select and purchase latest digital tools

2. **Flexible pricing:** Platform offers flexible pricing plans.

3. **Instant Access:** After buying, user can access the tools immediately
